# Android APK Nasıl İndirilir?

Bilgisayarınıza program kurmadan, GitHub üzerinden APK almak için adımlar:

## 1. Hazırlık
1. **[github.com](https://github.com)** adresine girin, üye olun/giriş yapın.
2. Sağ üst "+" -> **"New repository"**.
3. İsim olarak **`insaat-app`** yazın ve **"Create repository"** butonuna basın.

## 2. Dosya Yükleme (ÖNEMLİ!)
1. "uploading an existing file" linkine tıklayın.
2. Bilgisayarınızdaki **`ANTIGRA`** klasörünü açın.
3. İçindeki **TÜM** dosyaları seçip tarayıcıya sürükleyin.
   
   ⚠️ **DİKKAT:** `.github` isimli klasörün de yüklendiğinden EMEİN OLUN. 
   Eğer bu klasör yüklenmezse APK oluşmaz.

4. **"Commit changes"** butonuna basın.

## 3. APK İndirme
1. Sayfanın üzerindeki **"Actions"** sekmesine tıklayın.
2. **"Build Android APK"** yazısını görüp tıklayın.
   - *Eğer görmüyorsanız, `.github` klasörü yüklenmemiştir. Dosyaları tekrar yüklemeyi deneyin.*
3. İşlem bitince (Yeşil tik ✅), sayfanın altından **Artifacts** kısmındaki `insaat-asistani-apk` dosyasını indirin.

## 4. Kurulum
- İnen zipten çıkan `.apk` dosyasını telefona atıp kurun.
